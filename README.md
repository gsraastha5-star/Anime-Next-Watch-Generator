# Anime-Next-Watch-Generator
Project Description
-Anime Next Watch Generator is a web application designed to help users discover what anime to watch next. It fetches real-time data from a public API and allows users to explore trending anime, filter them by genre, and sort them based on ratings.

# API Used
Jikan API (https://docs.api.jikan.moe)

# Features
-View trending anime

-Filter anime by genre

-Search anime by name

-Sort anime by rating

-View detailed synopsis for each anime

# Technologies Used
1.HTML
2.CSS
3.JavaScript (Fetch API)
4.Array methods (map, filter, sort)

# How to Run
1.Clone the repository
2.Open index.html in your browser

# Purpose
This project demonstrates API integration, JavaScript concepts, and UI design skills.
